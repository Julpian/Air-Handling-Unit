## Air Handling Unit Inspection System

Web-based application developed for PT Kimia Farma Plant Banjaran Bandung to manage AHU inspection workflows including scheduling, digital inspection forms, NFC-based validation, and reporting.

### Features
- AHU inspection scheduling
- Digital inspection forms
- NFC integration
- Inspector assignment
- Inspection status tracking
- Reporting system

### Tech Stack
- Frontend: Next.js + Tailwind
- Backend: Golang (Gin)
- Database: PostgreSQL
