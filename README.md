# 🌬️ Air Handling Unit Inspection System

> Web-based inspection management system developed for **PT Kimia Farma Plant Banjaran Bandung** to digitize AHU inspection workflows — from scheduling to reporting — with NFC validation support.

---

## 🚀 Overview

This application streamlines **Air Handling Unit (AHU)** inspection processes by providing:

- Centralized inspection scheduling  
- Digital inspection forms  
- NFC-based validation  
- Inspector assignment  
- Real-time inspection status tracking  
- Structured reporting system  

Designed to improve operational efficiency, data accuracy, and traceability of AHU maintenance activities.

---

## ✨ Key Features

✅ AHU Inspection Scheduling  
📝 Digital Inspection Forms

## 🏗 Architecture

- Frontend: Next.js App Router + Tailwind UI
- Backend: Golang REST API (Gin Framework)
- Authentication: Token-based
- Database: PostgreSQL
- Integration: NFC for inspection validation

---

## 🎯 Purpose

Built to support **facility maintenance operations** at PT Kimia Farma Plant Banjaran Bandung by replacing manual inspection workflows with a digital, traceable, and scalable system.
